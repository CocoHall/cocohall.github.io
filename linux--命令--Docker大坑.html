Docker大坑



镜像和容器的区别：

镜像是静态资源，不可写
容器是镜像的实例化，镜像运行后会会产生一个容器


docker在linux下没有虚拟化，localhost就是主机
在windows下会先虚拟化一台linux再在linux上运行，默认ip:192.168.99.100

docker image list 查看镜像

docker save 镜像id > /opt/file.tar 导出镜像
docker load < /opt/file.tar 导入镜像

docker ps 查看运行的容器
docker ps -a 查看所有容器


docker rmi imageID 删除镜像
docker rm containerID 删除容器


docker run -d -p 127.0.0.1:物理机端口:容器端口(/udp) imageID
运行镜像，-d 后台运行，-p端口映射，前面的端口是主机端口
docker exec -it containerID /bin/bash 展示一个容器的shell页面

docker start containerID 运行一个容器


-e MAIL_PASSWORD='xxxxxx' 添加环境变量
-v $PWD/srcpm:/opt/webapp/srcpm 添加主机的目录，不能带空格，win下好像会出错
