启动IIS


命令行：

启动 net start w3svc
关闭 iisreset /stop

解释如下：


启动IIS
net start w3svc
启动两个服务
Windows Process Activation Service ( WAS )
World Wide Web Publishing Service ( W3SVC )

如果输入
iisreset /start
则只启动WAS,不能访问网页

关闭IIS
iisreset /stop

关闭两个服务
Windows Process Activation Service ( WAS )
World Wide Web Publishing Service ( W3SVC )

如果输入
net stop w3svc
只关闭W3SVC
