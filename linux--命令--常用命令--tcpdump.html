tcpdump -i dna0 -G 3600 -C 1024 host 183.131.12.206 -w ipv4_%Y%m%d%H%M%S.pcap

tcpdump -i any -w flow.pcap
抓取所有流量

tcpdump -i eth0 [src|dst|空] host|port|net IP地址或网络号 > 输出文件

src 源
dst 目的
不填 都抓取

host 主机IP
port 端口
net 网络 后面加网络号 如192.168


协议
tcpdump -i eth0 arp|ip|tcp|udp|icmp



非 : ! or "not" (去掉双引号)
与 : && or "and" 条件用()包含起来
或 : || or "or"

如
tcpdump -i eth1 '((tcp) and (port 80) and ((dst host 192.168.1.254) or (dst host 192.168.1.200)))'











- 只抓 SYN 包
# tcpdump -i eth1 'tcp[tcpflags] = tcp-syn'
- 抓 SYN, ACK
# tcpdump -i eth1 'tcp[tcpflags] & tcp-syn != 0 and tcp[tcpflags] & tcp-ack != 0'
抓 SMTP 数据
----------
# tcpdump -i eth1 '((port 25) and (tcp[(tcp[12]>>2):4] = 0x4d41494c))'
抓取数据区开始为"MAIL"的包，"MAIL"的十六进制为 0x4d41494c。
抓 HTTP GET 数据
--------------
# tcpdump -i eth1 'tcp[(tcp[12]>>2):4] = 0x47455420'
"GET "的十六进制是 47455420
抓 SSH 返回
---------
# tcpdump -i eth1 'tcp[(tcp[12]>>2):4] = 0x5353482D'
"SSH-"的十六进制是 0x5353482D


# tcpdump -i eth1 '(tcp[(tcp[12]>>2):4] = 0x5353482D) and (tcp[((tcp[12]>>2)+4):2]
= 0x312E)'抓老版本的 SSH 返回信息，如"SSH-1.99.."



- 抓 DNS 请求数据
# tcpdump -i eth1 udp dst port 53


用-c 参数指定抓多少个包
