netsh端口转发


windows 7上就可以直接用netsh了，在xp/2003的话，要先安装ipv6

netsh interface ipv6 install

将本地端口443转发至192.168.1.200的3389端口

netsh interface portproxy add v4tov4 listenport=443 listenaddress=0.0.0.0 connectaddress=192.168.1.200 connectport=3389


可使用命令netsh interface portproxy show all来查看所有已建转发规则
PS：删除转发规则 ，add改为delete，注意：防火墙例外443端口
