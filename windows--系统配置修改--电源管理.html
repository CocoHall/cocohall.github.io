电源管理


SETACVALUEINDEX：修改交流电的设置
SETDCVALUEINDEX：修改电池的设置

获取当前的电源方案UUID
powercfg /GETACTIVESCHEME

睡眠按钮：不采取任何操作
powercfg /SETACVALUEINDEX uuid SUB_BUTTONS SBUTTONACTION 0
powercfg /SETDCVALUEINDEX uuid SUB_BUTTONS SBUTTONACTION 0

电源按钮：不采取任何操作
powercfg /SETACVALUEINDEX uuid SUB_BUTTONS PBUTTONACTION 0
powercfg /SETDCVALUEINDEX uuid SUB_BUTTONS PBUTTONACTION 0

合上盖子：不采取任何操作
powercfg /SETACVALUEINDEX uuid SUB_BUTTONS LIDACTION 0
powercfg /SETDCVALUEINDEX uuid SUB_BUTTONS LIDACTION 0

在此时间后关闭显示：从不
powercfg /SETACVALUEINDEX uuid SUB_VIDEO VIDEOIDLE 0
powercfg /SETDCVALUEINDEX uuid SUB_VIDEO VIDEOIDLE 0

在此时间后睡眠：从不
powercfg /SETACVALUEINDEX uuid SUB_SLEEP STANDBYIDLE 0
powercfg /SETDCVALUEINDEX uuid SUB_SLEEP STANDBYIDLE 0

在此时间后休眠：从不
powercfg /SETACVALUEINDEX uuid SUB_SLEEP HIBERNATEIDLE 0
powercfg /SETDCVALUEINDEX uuid SUB_SLEEP HIBERNATEIDLE 0

不休眠
powercfg /x hibernate-timeout-ac 0
不睡眠
powercfg /x standby-timeout-ac 0
