autorunsc


查看开机启动项 命令行

-a 选择查看项:
* 所有
b Boot execute 只能调用Native API函数 HKLM\System\CurrentControlSet\Control\Session Manager\BootExecute
d Appinit DLLs. USER32_DLL 会调用 HKEY_LOCAL_MACHINE\Software\Microsoft\WindowsNT\CurrentVersion\Window\Appinit_Dlls，HKEY_LOCAL_MACHINE\Software\Microsoft\WindowsNT\CurrentVersion\Window\LoadAppInit_DLLs
e Explorer addons. 资源管理器加载项、启动等
g Sidebar gadgets (Vista and higher) 桌面的工具栏
h Image hijacks. 后缀的打开方式等
i Internet Explorer addons. IE加载项
k Known DLLs. 已知DLL
l Logon startups (this is the default). 登陆脚本
m WMI entries. WMI开机启动
n Winsock protocol and network providers. 网络相关
o Codecs. office
p Printer monitor DLLs. 打印相关的
r LSA security providers. 内置的安全设施
s Autostart services and non-disabled drivers. 服务、驱动
t Scheduled tasks. 计划任务
w Winlogon entries.
输出方式
-c Print output as CSV.
-ct Print output as tab-delimited values.
-x Print output as XML.

哈希
-h Show file hashes.

-m Hide Microsoft entries (signed entries if used with -v). 隐藏微软签名
-s Verify digital signatures. 验证数字签名
-t Show timestamps in normalized UTC (YYYYMMDD-hhmmss). 显示时间戳

使用VirusTotal检测或只显示未签名的
-u If VirusTotal check is enabled, show files that are unknown by VirusTotal or have non-zero detection, otherwise show only unsigned files.

文件哈希和VirusTotal联动
-v[rs] Query VirusTotal for malware based on file hash. Add 'r' to open reports for files with non-zero detection. Files reported as not previously scanned will be uploaded to VirusTotal if the 's' option is specified. Note scan results may not be available for five or more minutes.
同意VirusTotal条款
-vt Before using VirusTotal features, you must accept the VirusTotal terms of service. If you haven't accepted the terms and you omit this option, you will be interactively prompted.
离线扫描
-z Specifies the offline Windows system to scan.

指定用户名
user Specifies the name of the user account for which autorun items will be shown. Specify '*' to scan all user profiles.
