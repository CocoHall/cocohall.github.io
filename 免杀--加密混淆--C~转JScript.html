我们需要创建一个 C# 工程，我这里使用 Visual Studio 2017。新建一个空白的 C# 的 Console 工程，.Net Framework 版本选择 2.0（保证兼容性）。

修改该工程的属性，将输出类型改为“Windows 应用程序”，启动对象改为“ConsoleApp3.Program”

代码里增加函数，Program
using System;
using System.Windows.Forms;
namespace ConsoleApp3
{
    public class Program
    {
        public Program()
        {
            MessageBox.Show("Program");
        }
        public static void Main()
        {
            MessageBox.Show("test");
        }
    }
}

然后生成exe文件，双击弹出test弹窗

执行命令
DotNetToJScript.exe -l=JScript -o=ConsoleApp3.js -c=ConsoleApp3.Program ConsoleApp3.exe

点击1.js弹出Program对话框


-l 参数可选择其他脚本：JScript, VBA,VBScript
--ver 表示c#的版本None, v2, v4, Auto
-c 表示函数入口
