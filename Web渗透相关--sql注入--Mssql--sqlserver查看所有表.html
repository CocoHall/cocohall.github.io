sqlserver查看所有表


命令行连接
sqlcmd -S localhost -U sa -P sa -d 数据库 -Q "查询语句" -o 输出文件
每行命令输完后输入go回车


1.获取所有数据库名:
SELECT Name FROM Master..SysDatabases ORDER BY Name

2.获取所有表名:
SELECT Name FROM 'DatabaseName'..SysObjects Where XType='U' ORDER BY Name

'DatabaseName' 替换为需要查询的数据库

XType='U':表示所有用户表;
XType='S':表示所有系统表;
TR 指的是触发器，

3.获取所有字段名:
SELECT Name FROM SysColumns WHERE id=Object_Id('TableName')


'TableName' 替换为需要查询的表



可以看数据库保存的路径
select * from dbo.sysdatabases

查看数据库状态
sp_helpdb
