修改时间戳


windows：
三种时间戳
LastWriteTime （属性修改不会更改这个时间）
CreationTime
LastAccessTime

Get-Childitem -path '.' -Recurse | foreach-object { $_.LastWriteTime = Get-Date; $_.CreationTime = Get-Date;$_.LastAccessTime = Get-Date;}

Get-Childitem -path '.' -Recurse | foreach-object { $_.LastWriteTime = '01/01/1970 00:00:01'; $_.CreationTime = Get-Date;$_.LastAccessTime = Get-Date;}

Get-Childitem -path '.' -Recurse | foreach-object { $_.LastWriteTime = '05/02/2010 19:31:30'; $_.CreationTime = '05/02/2010 19:31:30';$_.LastAccessTime = '05/02/2010 19:31:30';}

Get-Childitem -path '.' -Recurse | foreach-object { $_.LastWriteTime = '01/04/2011 16:27:27'; $_.CreationTime = '01/04/2011 16:27:27';$_.LastAccessTime = '09/13/2011 19:45:33';}

linux:
三种时间戳
mtime 修改内容时间
ctime 修改属性时间
atime 访问时间

复制 index.php 的时间，赋给 webshell.php
touch -r index.php webshell.php

修改访问时间和修改时间
touch -t 201708170708.25 file
touch -d "2013-10-12 10:21:34" file
