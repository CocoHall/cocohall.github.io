后台运行


1、nohup

nohup python test.py &

输入输出会重定向，后台运行，终端关闭后继续执行

2、setsid

setsid python test.py

没有重定向，终端看起来还是正常执行命令，终端关闭后继续执行

3、( python test.py &)

后台执行，终端关闭后继续执行

4、screen

screen -dmS session_name 创建新的session，不连接
screen -list 列出所有会话。
screen -r session_name 重新连接指定会话。
快捷键CTRL-a d 来暂时断开当前会话。

5、disown

场景：已经执行了耗时任务，需要用disown保持其连续执行

disown -h jobspec来使某个作业忽略HUP信号

使用jobs查看job

ctrl+z 放入后台，记录jobs号
bg jobs
disown -h jobs
