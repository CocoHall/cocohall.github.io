临时文件


php form-data 形式的文件上传会在临时目录生成一个临时文件

临时文件名
/tmp/phpxxxxxx
C:\Windows\phpxxxx.tmp

将文件post给phpinfo.php，会返回该临时文件名

如下是java利用脚本，大量冗余数据是为了使临时文件删除得慢一点

    public static String getData(InputStream is) throws Exception

    public static String getPhptmp(String data)

        }
        return tmp;
    }

// TODO Auto-generated catch block
